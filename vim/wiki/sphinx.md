# O que é

Sphinx é uma ferramenta para criar uma documentação de maneira fácil,
inteligente e bonita.

# Como instalar?

    pip install sphinx

# Inicializando a doc

     mkdir docs

     cd docs

     sphinx-quickstart

#ComPILANDO PARA HTML

     make html

# EXIBINDO NO BROWSER

     python -m SimpleHTTPServer 9000
