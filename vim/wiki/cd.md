# cd - mudar de diretórios - Last Change: qui 02 nov 2017 13:06:32 -03

    cd -    ............ voltar ao último diretório
    cd ..   ............ subir um nível
	cd      ............ quando digitado sem argumentos volta ao $HOME

Pra quem usa o `zsh` podemos fazer coisas assim:

		cd =python3(:h)
