Terminal multiplexer

# Shortcuts

		C-b %  ........... split window vertically
		C-b ;  ........... toggle last active pane
		C-b q  ........... show pane numbers
		C-b o  ........... which to next pane
		C-b x  ........... close current pane
		C-b ?  ........... show shortcuts
