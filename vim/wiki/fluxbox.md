# Criar pasta de backgrounds do fluxbox

ln -s ~/Dropbox/img/wallpapers/ ~/.backgrounds/

fbsetbg -r ~/.backgrounds/


Para que isto fique permanente faca:

vim ~/.fluxbox/init

    session.screen0.rootCommand: fbsetbg -r ~/.backgrounds/
